# Preset Inspirations

The current showcase canvas is a single original reinterpretation informed by several public PyQt6 ecosystems. It does not reproduce upstream layouts, assets, or code verbatim.

- `UI Showcase Playground`
  - Inspired by `zhiyiYo/PyQt-Fluent-Widgets`, `pyqtgraph/pyqtgraph`, `pyapp-kit/superqt`, `5yutan5/PyQtDarkTheme`, and `Wanderson-Magalhaes/Modern_GUI_PyDracula_PySide6_or_PyQt6`.
  - Focus: simple controls, dense nested shells, flyouts, dialogs, tables, tree inspectors, scroll regions, styling studies, and effect tuning on one giant canvas.
